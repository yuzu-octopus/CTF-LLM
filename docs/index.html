CTF-LLM

Fine-tune LLMs to solve CTF challenges

End-to-end pipeline for fine-tuning open-source LLMs (Gemma 4, Qwen 3.5) on cybersecurity CTF challenges using Unsloth + QLoRA on Google Colab's free T4 GPU.

View on GitHub Quick Start

Features

End-to-end Pipeline

Scrapes CTF writeups from 13 GitHub repos, downloads curated HuggingFace datasets, processes to ChatML, trains with Unsloth QLoRA, evaluates on a 210-question benchmark.

4 Models Supported

Gemma 4 E4B, Gemma 4 12B, Qwen 3.5 9B, Qwen 3.5 4B — all running 4-bit QLoRA on a free T4 GPU (16GB VRAM).

210-Question Benchmark

Curated evaluation set across 4 categories (pwn, rev, crypto, web) and 3 difficulty levels with Wilson 95% CI, McNemar's paired test, and contamination checking.

Colab-Ready

Single finetune.sh handles session creation, dependency install, file upload, training, and model download — zero manual Colab UI interaction.

Benchmark

210 questions across 4 categories and 3 difficulty levels.

  • Categories — pwn (57), rev (50), crypto (50), web (53)
  • Difficulties — easy (72), medium (79), hard (59)
  • Task types — flag_extraction, multiple_choice, code_generation, vulnerability_identification, patch_generation, exploit_trace
  • Statistical rigor — Wilson 95% CI (±6.8%), McNemar's test, contamination check, cheating detection

Quick Start

# Install dependencies
uv sync

# Run full pipeline (build data + train)
./finetune.sh gemma4 --all

# Evaluate trained model
./finetune.sh gemma4 --eval
